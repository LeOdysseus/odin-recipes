Simple Recipe Website

Project Overview

This project features a main index.html page that serves as a simple navigation hub, linking to individual recipe pages through different links. 

Skills demonstrated

Once finished, this project will show:
-The capacity to build a basic HTML structure with <!DOCTYPE html>, <html>, <head>, and <body> tags.

-Using headings (<h1> to <h6>), paragraphs (<p>), and lists (<ul> or <ol>) to present content in a good structure.

-Implementing links (<a href="">) to connect pages and navigate between recipes.

-Structuring a multi-page site with separate HTML files in a logical folder setup.
